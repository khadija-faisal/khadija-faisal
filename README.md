## What I'm Building

### [Nuralyxai](https://www.nuralyxai.com) — Agentic AI Assistant for Freelancers

Freelancers lose 10+ hours a week on admin — emails, proposals, meetings, follow-ups. Nuralyxai handles all of it through one chat interface. You tell it what you need. It handles the rest.

**The Architecture:**

```
User Message
     ↓
Orchestrator Agent  (routes intent to the right agent)
     ↓
┌─────────────────────────────────────────────────┐
│  Email Agent       → reads threads, drafts replies    │
│  Follow-up Agent   → detects silence, nudges clients  │
│  Proposal Agent    → turns job posts into proposals   │
│  Calendar Agent    → schedules, updates, cancels      │
│  Reminder Agent    → sets push/email/voice alerts     │
└─────────────────────────────────────────────────┘
     ↓
Context carried across every turn — no repetition
```

> 500+ files. Production-grade. Built from scratch as a student.
