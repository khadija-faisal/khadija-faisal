## What I'm Building

**[Nuralyxai](https://www.nuralyxai.com)** — an agentic AI assistant that automates the full client workflow for freelancers through one chat interface. Emails, proposals, meetings, follow-ups, and reminders — all context-aware, all connected.

I built the entire backend solo from scratch.

```
Multi-agent orchestration  →  OpenAI Agents SDK
5 specialized AI agents    →  Email · Follow-ups · Proposals · Calendar · Reminders  
Google integrations        →  Gmail API · Google Calendar API · OAuth 2.0
Security layer             →  JWT Auth · Fernet Encryption · PII Sanitization
Infrastructure             →  FastAPI · SQLAlchemy · Alembic · Uvicorn
```

> 100+ files. Production-grade. Built while being a student.
